###REQUERIMIENTOS DEL PROYECTO.

• El servicio web ser desarrollada en NestJS (sin utilizar frameworks o
plataformas adicionales). 
• Aplicar base de datos sobre MongoDb o PostgreSQL. 
• Presentar una capa de servicios bien estructurada. 
• Debe responder a las necesidades del servicio según tema escogido por
el estudiante. 
• Aplicar relaciones en su modelo de base de datos (consultar el modelo
más adecuado según su proyecto de requerir utilizar relacional o no
relacional). 
• Generar documentación del servicio web utilizando Nestjs swagger -
OpenAPI Specification
